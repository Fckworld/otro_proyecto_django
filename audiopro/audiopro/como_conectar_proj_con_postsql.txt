CONTEXTO:
    LOS PASOS DESCRIBTOS EN MODELOS.PY SON LOS PASOS PARA CREAR 
    LA BASE DE DATOS CON SQLLITE, QUE VIENE POR DEFECTO EN DJANGO.
    AQUI SE MOSTRARA COMO CONECTAR EL PROYECTO CON LA BASE DE DATOS 
    POSTGRESQL.

1) INSTALAR PAQUETE PSYCOPG2: pip install psycopg2
2) CONFIGURAR SETTING.PY EN DATABASE PARA POSTGRESQL
3) NO OLVIDAR INSTALL GESTOR DE POSTGRESQL, CLAVE DESIGNADA:admin
   PUERTO DESIGNADA: 5432
   EJEMPLO:
        'USER':'postgres',
        'PASSWORD':'admin',
        'HOST':'127.0.0.1',
        'DATABASE_PORT':'5432',
4)UNA VEZ CREADA LA BASE DE DATOS EN POSTGRESQL, NECESITO HACER LAS MIGRACIONES CORRESPONDIENTES
OSEA, CREAR LAS TABLAS DE LOS MODELOS EN POSTGRESQL.
    EJECUTAMOS:
        (ESTO LO EJECUTAMOS CADA VEZ QUE HACEMOS UN CAMBIO EN LA BASE DE DATOS)
        python manage.py makemigrations /* DETECTA LOS CAMBIOS REALIZADOS EN LOS MODELOS*/
        python manage.py migrate /* ESTE ES EL IMPORTANTE*/
